알림

앱 내 알림(Notification)과 브라우저 웹 푸시(PushSubscription)를 함께 관리합니다. web-push 패키지와 VAPID 키를 사용합니다.

마지막 업데이트 2026-05-14

📋 개요

  • → 앱 내 알림: Notification 모델 — 연차 승인/반려, 일정 초대 등 저장
  • → 웹 푸시: Service Worker + PushSubscription 모델 — 브라우저 백그라운드 수신
  • → /notification — 알림 목록 페이지
💡

앱 내 알림과 웹 푸시는 독립적으로 동작합니다. 웹 푸시를 허용하지 않아도 앱 내 알림은 정상 표시됩니다.

lib/push.ts

TSsrc/lib/push.ts
import webpush from "web-push"
import { db } from "@/lib/db"

webpush.setVapidDetails(
  "mailto:admin@togate.com",
  process.env.VAPID_PUBLIC_KEY!,
  process.env.VAPID_PRIVATE_KEY!
)

interface PushPayload {
  title: string
  body:  string
  url?:  string
}

export async function sendPushToAll(payload: PushPayload) {
  const subs = await db.pushSubscription.findMany()
  const results = await Promise.allSettled(
    subs.map(sub =>
      webpush.sendNotification(
        { endpoint: sub.endpoint, keys: { p256dh: sub.p256dh, auth: sub.auth } },
        JSON.stringify(payload)
      )
    )
  )

  // 만료된 구독(410) 자동 삭제
  const expired = results
    .map((r, i) => ({ r, sub: subs[i] }))
    .filter(({ r }) => r.status === "rejected" &&
      (r.reason?.statusCode === 410 || r.reason?.statusCode === 404))

  if (expired.length) {
    await db.pushSubscription.deleteMany({
      where: { id: { in: expired.map(({ sub }) => sub.id) } },
    })
  }
}

Service Worker (public/sw.js)

JSpublic/sw.js
self.addEventListener("push", (event) => {
  const data = event.data?.json() ?? {}
  event.waitUntil(
    self.registration.showNotification(data.title ?? "Togate", {
      body: data.body,
      data: { url: data.url },
    })
  )
})

self.addEventListener("notificationclick", (event) => {
  event.notification.close()
  const url = event.notification.data?.url
  if (url) event.waitUntil(clients.openWindow(url))
})

🗄️ DB 모델

TSprisma/schema.prisma — Notification · PushSubscription
model Notification {
  id        String   @id @default(cuid())
  userId    String
  message   String
  url       String?
  isRead    Boolean  @default(false)
  createdAt DateTime @default(now())
  user      User     @relation(fields: [userId], references: [id])
}

model PushSubscription {
  id       String @id @default(cuid())
  userId   String
  endpoint String @unique
  p256dh   String
  auth     String
  user     User   @relation(fields: [userId], references: [id])
}
