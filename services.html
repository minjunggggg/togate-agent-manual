🔔 웹 푸시 VAPID 키

VAPID 키는 외부 서비스가 아닌 프로젝트 자체적으로 생성합니다. 키를 분실하거나 재생성하면 기존 구독자의 푸시가 모두 끊기므로 주의가 필요합니다.

🚨

VAPID 키 재생성 금지: 키를 바꾸면 DB의 모든 PushSubscription 레코드가 무효화되어 전 직원이 푸시 알림을 받을 수 없게 됩니다. 재생성이 불가피한 경우 알림 재구독 안내를 사전에 공지해야 합니다.

항목내용
Public KeyVercel 환경변수 NEXT_PUBLIC_VAPID_PUBLIC_KEY에 저장
Private KeyVercel 환경변수 VAPID_PRIVATE_KEY에 저장
키 생성 명령어npx web-push generate-vapid-keys
키 백업 위치[ 기입 필요 — 팀 보안 채널 ]

⚡ 사용량 & 한도

외부 서비스의 무료 한도 또는 요금제 한도를 초과하면 서비스가 중단되거나 추가 요금이 발생합니다.

서비스주요 한도초과 시 증상모니터링
Gemini AI분당 요청 수 (모델별 상이)회의록 생성 실패 (429 에러)Google AI Studio → Usage
Deepgram월 음성 전사 시간 또는 크레딧음성 전사 실패Deepgram Console → Usage
OpenAI월 크레딧 한도API 호출 실패 (429 에러)platform.openai.com → Usage
Vercel월 빌드 시간, 함수 실행 시간, 대역폭배포 실패 또는 함수 타임아웃Vercel 대시보드 → Usage
DB스토리지, 연결 수연결 거부 또는 쓰기 실패DB 서비스 대시보드
💡

비용 알림 설정 권장: 각 서비스의 대시보드에서 예산 알림을 설정해두면 예상치 못한 비용 급증을 조기에 감지할 수 있습니다.
