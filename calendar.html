캘린더

월별 캘린더 뷰에서 팀 일정과 연차를 함께 확인합니다. 직원은 이 페이지에서 연차를 신청할 수 있습니다.

마지막 업데이트 2026-05-14

📋 개요

  • → /calendar — 이번 달 캘린더 (Schedule + LeaveRequest 표시)
  • → /calendar/new — 연차 신청 폼
  • → /calendar/[id] — 연차 신청 상세
💡

캘린더는 Schedule(팀 일정)과 LeaveRequest(연차)를 함께 표시합니다. 연차 승인 시 leaveRequestId가 연결된 Schedule이 자동 생성됩니다.

TSapp/api/leave-requests/route.ts — POST
export async function POST(req: Request) {
  const session = await auth()
  const { type, date, reason } = await req.json()

  // 잔여 연차 확인
  const user = await db.user.findUnique({ where: { id: session!.user.id } })
  const cost = type === "ANNUAL" ? 1 : 0.5
  if ((user!.annualLeave ?? 0) < cost)
    return NextResponse.json({ error: "잔여 연차가 부족합니다." }, { status: 400 })

  const leave = await db.leaveRequest.create({
    data: {
      type, date: new Date(date), reason,
      userId:  session!.user.id,
      status:  "PENDING",
    },
  })

  // ADMIN에게 앱 알림
  const admins = await db.user.findMany({ where: { role: "ADMIN" } })
  await db.notification.createMany({
    data: admins.map(a => ({
      userId:  a.id,
      message: `연차 신청이 접수되었습니다.`,
      url:     `/admin/leave/${leave.id}`,
    })),
  })

  return NextResponse.json(leave, { status: 201 })
}

🗄️ DB 모델

TSprisma/schema.prisma — LeaveRequest
model LeaveRequest {
  id         String      @id @default(cuid())
  userId     String
  type       LeaveType               // ANNUAL | HALF_DAY_AM | HALF_DAY_PM
  date       DateTime
  reason     String?
  status     LeaveStatus @default(PENDING) // PENDING | APPROVED | REJECTED
  user       User        @relation(fields: [userId], references: [id])
  schedules  Schedule[]              // 승인 시 Schedule 자동 생성
}

enum LeaveType   { ANNUAL  HALF_DAY_AM  HALF_DAY_PM }
enum LeaveStatus { PENDING  APPROVED  REJECTED }

⏰ Cron — 연차 초기화

매년 12월 31일 15:00 UTC(한국 1월 1일 0시)에 전 직원의 annualLeave를 15일로 초기화합니다.

TSapp/api/cron/reset-annual-leave/route.ts
export async function GET(req: Request) {
  const auth = req.headers.get("Authorization")
  if (auth !== `Bearer ${process.env.CRON_SECRET}`)
    return NextResponse.json({ error: "Unauthorized" }, { status: 401 })

  await db.user.updateMany({
    where: { role: "EMPLOYEE" },
    data:  { annualLeave: 15 },
  })

  return NextResponse.json({ ok: true })
}
