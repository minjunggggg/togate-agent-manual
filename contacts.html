비상연락망

직원 연락처 디렉토리입니다. Employee 모델을 기반으로 이름·전화번호·직책을 조회하고 관리합니다.

마지막 업데이트 2026-05-14

📋 개요

  • → /contacts — 전체 직원 목록 (검색·필터)
  • → /contacts/[id] — 직원 상세 정보
  • → /contacts/[id]/edit — 직원 정보 수정 (ADMIN 전용)
💡

모든 로그인 사용자가 목록·상세를 조회할 수 있습니다. 수정은 ADMIN만 가능합니다.

TSapp/api/contacts/route.ts
import { NextResponse } from "next/server"
import { db } from "@/lib/db"

export async function GET() {
  const employees = await db.employee.findMany({
    include: { user: { select: { email: true, role: true } } },
    orderBy: { name: "asc" },
  })
  return NextResponse.json(employees)
}

🗄️ DB 모델

TSprisma/schema.prisma — Employee
model Employee {
  id       String  @id @default(cuid())
  name     String           // 이름
  phone    String?          // 전화번호 (선택)
  position String           // 직책
  userId   String  @unique
  user     User    @relation(fields: [userId], references: [id])
}
필드타입설명
nameString직원 이름 (필수)
phoneString?전화번호 (선택, null 가능)
positionString직책 (필수)
userIdString @uniqueUser와 1:1 연결
